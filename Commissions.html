Garry's Mod Scripting

I am experienced with all the libraries that Garry's Mod comes with and I have experience in all areas as I've had to dive into everything at least once thanks to the MMM project. Whatever you desire, I can make it a reality.*

     • Administrative scripts. (Commands, player control & permissions, etc..)
     • Security related scripting. (Anti-cheat, anti-crash, prop protection, module injection, etc..)
     • Economy related scripting. (Pointshops, NPC traders, currency management, economy balancing, Inventory systems, etc..)
     • Player management related scripting. (Player save data manager, player tags, custom nicknames, custom nick colors, etc..)

     • NPC & AI scripting.
     • HUD / UI design.
     • Derma, vgui / vgui menu design.
     • 3D2D / 2D Rendering and projection. (Example: Rendering 2D text in 3D space)
     • Making old scripts compatible with gmod 11.
     • Performance overhauling / optimizing scripts.
     • Porting weapons from one base to another. (may be limited)
     • Porting scripts. (may be limited)

Benefits:

     • All scripts that are made by me will be as optimized as possible to have as little impact on the server as possible.*
          * (Within my capabilities, there is ALWAYS a way to improve something further.)

Discord Bot Creation

I can make a custom discord bot for your community or private server.
The coding language used would be Lua using the Discordia library.

C++, javaScript, and Python are the most common coding languages to make discord bots with.
Lua and the Discordia API have their limits, not everything can be made a reality.
It can handle all discord related things, however, non-discord related things are limited.

Requirements:
     • You're able to create a new discord application.
     • You're able to supply the auth token of the created discord application.
          See: https://discord.com/developers/applications

Limitations:
     • Luvit A-Sync only works on specific platforms. These being Windows, Linux x86 and x64, and Raspberry Pi OS (armv6l and armv7l).
          - Binaries for other platforms do exist, however I cannot guarantee that they will work.
               See: https://github.com/truemedian/luvit-bin/releases for all available binaries.

Benefits:
     • You will have 1 year of support on everything I make. If you encounter issues, I will try to resolve them. Free of charge.

I won't do any of the following: 
     • Database related things. (Exception: Pseudo-databases that store information in plain text files)
     • Garry's Mod related integrations.

Hosting:

     ​Just like every other always-online service, the discord bot has to be hosted on a server.
          (Unless you're fine with it not being available at all times and want to host it locally on your computer)

     Using a VPS (Virtual Private Server) running ubuntu or debian is recommended. Below is a list of hosting providers.
          Note: The providers listed have been found through public articles from other people. I cannot personally recommend them as I have no experience with them. Use them at your own risk.

          Easy to set up:
               - SomethingHost
               - xGamingServer

          VPS (Virtual Private Server) hosts:
               - Vultr
               - Amazon EC2
               - Kamatera

     Hosting the bot on local hardware:
          If you own a Raspberry Pi 3 or newer, you can host the Discord bot locally.
          This requires Raspberry Pi OS for ARM processors (Debian ARM x86) and a custom Luvit binary.
               See: https://github.com/truemedian/luvit-bin/releases (armv6l or armv7l).

     If you don't know how or cannot be bothered to set up the bot hosting:
          I can set up the hosting of the bot for you. (Exception: Raspberry Pi)
          Depending on the service provider that you choose; I will charge an additional 5 to 15 euros for this.

Pricing:
     ​Pricing starts at 20€
