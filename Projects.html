A modern and powerful program to easily change the configuration files for SMITE PC on Windows.
Fully compatible with Windows 7, Windows 8, and Windows 10.

A popular choice when it comes to increasing FPS in the game SMITE.
It can make the game run on almost any potato out there.

Developed in 2017 and still being maintained to this day, an open source project that helped countless of people
to be able to run the game on their computer and is able to fix stuttering and freezing on high-end computers!


  Officially recognized by Hi-Rez Studios 

The SMITE Optimizer was officially recognized to be an "outstanding contribution" to the community.
> https://ibb.co/album/bbLhkQ <


 Simple to use & User friendly​   

Designed to be as simple to use and user friendly as possible.
You can hover over each setting to receive information about what it does through popups.
If you don't want to mess with anything yourself you can simply apply the "High performance settings"
The program creates a backup of your configuration files so that you can undo any changes when you're not satisfied with the result.


 Powerful & Useful extras​   

'Uncapping' the FPS, finally you can play with more than 150 FPS!
Fixing audio ques cutting out during larger team fights. (By increasing the amount of Audio Channels)


Using the Standalone Launcher post 9.4 patch ​   

This program is able to install the standalone (legacy) launcher, however it no longer receives game updates making it obsolete. Eventually Hi-Rez Studios is going to take down the update servers, meaning you wont be able to use it at all.

This is why a new method has been implemented into the SMITE Optimizer. It is called the "Quicklaunch Bypass"
It is using the Steam or Epic Game Store version of the game and launches it like the standalone launcher does.
You still receive all benefits of the standalone launcher using this method including being able to run the game without Steam or the Epic Game Store open as well as a major performance increase.
